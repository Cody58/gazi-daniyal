### ⚡ Arithmetic Circuit Design
> *Logic Design — November 2025*

Designed and simulated a complete series of arithmetic circuits in **Quartus Prime** using hierarchical and modular design methodology.

- 🔹 1-bit Full Adder — designed from scratch using fundamental logic gates
- 🔹 4-bit Adder/Subtractor — with **carry** and **overflow** detection flags
